# Capstone-Project
****Bank customer Churn Analysis- Completed as a part of certification program. ****

• Analysed 10,000 retail bank customer records to identify key drivers of churn.  

• Performed end-to-end data cleaning, interpretation, identifying patterns and trends. 

• Built interactive dashboard highlighting KPIs for churn rate, age group, product segment. 

• Applied SQL for deep-dive querying churn rate by product, Geographical distribution and tenure of 
customers, Credit card ownership, Cibil score and customer risk identification. 

• Delivered insights about churn behaviour and retention strategies.
